# Compendium Workouts

## Summary

Extend the compendium to include **workouts** — curated, public workout templates that any user can browse and import into their personal "my workouts." Valuable standalone (shared workout library) and a prerequisite for other social features like workout groups.

A compendium workout is composed of:

- **Name** and **description**
- An ordered list of **exercises** (referencing compendium exercises), each with prescribed sets/reps/weight/duration/etc.

When a user "imports" a compendium workout, it creates a personal copy (my-workout) linked to the compendium source. The user's copy is independent — they can modify it without affecting the original.

## Considerations

- Who can create compendium workouts? Only admins, or any user can publish?
- Should there be versioning? If the compendium workout is updated, should importers be notified?
- Tagging/categorization for discoverability (e.g., "push/pull/legs", "full body", "beginner").
